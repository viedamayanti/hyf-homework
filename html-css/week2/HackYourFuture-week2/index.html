Web Development As An Industry

Examples of dramatic transformation in communication and commerce led by web development include e-commerce. Online auction sites such eBay have changed the way consumers find and purchase goods and services. Obline retailers such as Amazon.com and Buy.com (among many others) have transformed the shopping and bargeain-hunting experience for many consumers.

Practical Web Development

Since the commercialization of the web, web development has been growing industry. The growth of this industry is being driven by businesses wishing to use their website to sell products and service to costumers. [1]

There is open source software for web development like BerkeleyDB, GlassFish, LAMP (Linux, Apache, MySql, PHP) stack and Perl/Plack. This has kept cost of learning web development to a minimum. Another contributing factor to the growth of the industry has been the rise of easy-to-use WYSIWYG web-developmentsoftware, such as Adobe Dreamweaver, BlueGriffon and Microsoft Visual Studio. Knowledge of HyperText Markup Language (HTML) or of programming language is still required to use such software, but the basics can be learned and implemented quickly with the help of help files, tecnical books, internet tutorials, or face-to-face training.

An ever-growning set of tools and technologies have helped developers build more dynamic and interactive websites. Further, web developers now help to deliver applications as web service which were traditionally only available as applications on a desk-based computer. This has allowed for many opportunities to decentralize information and media distribution. Examples can be seen with the rise of cloud service such as Adobe Creative Cloud, Dropbox and Google Docs. These web service allow users to interact with applications from many locations, instead of being tied to a specifik workstation for their application environment
